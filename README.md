## plusone英语站手机网站 + 后台管理

### 技术栈
vue + element + koa2 + sequelize(MySQL)

### 功能
1. 网页展示
2. 用户注册登录
3. 打卡，收藏，测试
4. 后台信息管理

### run fontend
+ npm install
+ npm run dev

### run server
+ cd backend
+ npm install
+ npm run dev
