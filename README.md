## plusone移动端网站 + 后台管理

### 技术栈
`vue + element + koa2 + sequelize(MySQL)`

### 功能
- 移动端网页展示
- 用户注册登录
- 打卡，收藏，测试
- 后台信息管理

### run fontend
```js
npm install
npm run dev
```
### run server
```js
cd backend
npm install
npm run dev
```
### 特殊功能（开发中）
- 主页布局拖拽式设置（完善功能ing）
- 测试试卷（完善功能ing）
